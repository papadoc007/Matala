# מטלה 04 — ארכיטקטורת סוכני AI (מסמך LaTeX)

## מה זה בכלל?

זה הפרויקט להגשה **מטלה 4** בקורס **AI Agents** (אוניברסיטת בר-אילן).  
המטלה לא הייתה "רק לכתוב עבודה" — היא בדקה שאנחנו יודעים:

1. **להבין** את ארכיטקטורת סוכני AI (כמו שנלמד בהרצאה 7)
2. **לייצר מסמך PDF מקצועי** בעברית עם מונחים באנגלית
3. **לעבוד נכון עם LaTeX** — קובץ תוכן (`main.tex`) נפרד מקובץ עיצוב (`agentic-ai-template.cls`)
4. **לתעד את העבודה** כמו בפרויקט תוכנה: `PRD.md`, `PLAN.md`, `TODO.md`

בקיצור: בנינו **ספר אקדמי שלם** על נושא הסוכנים, והפקנו ממנו PDF של **32 עמודים**.

## איך מקמפלים מחדש את ה-PDF?

### דרישות
- **MiKTeX** מותקן (כולל `lualatex`)
- Windows עם פונט **Arial** (לעברית)

### הפקודה הפשוטה

```powershell
cd metal4
.\compile.ps1
```

### אם MiKTeX שואל כל הזמן "להתקין חבילה?"

זה קרה גם אצלנו. הפתרון:

1. בטלו את הסימון **"Always show this dialog"** בחלון ההתקנה
2. הריצו פעם אחת:
   ```powershell
   initexmf --set-config-value "[MPM] AutoInstall=1"
   initexmf --update-fndb
   ```
3. השתמשו ב-`compile.ps1` — הוא כבר מריץ עם `--enable-installer`

## איך זה נבנה בפועל?

1. **קראנו את תמלול ההרצאה** — זה המקור העיקרי לתוכן
2. **הגדרנו PRD** — מה בדיוק צריך להיות במסמך (30+ עמודים, CLS, ביבליוגרפיה...)
3. **כתבנו `agentic-ai-template.cls`** — כל העיצוב: עברית RTL, `\eng{}` לאנגלית, כותרות, תיבות, header/footer
4. **כתבנו 15 פרקים** ב-`chapters/` והרכבנו אותם ב-`main.tex`
5. **קימפלנו** עם LuaLaTeX + BibTeX עד שיצא `main.pdf` תקין

הרעיון מהשיעור: **הפרדה בין תוכן לעיצוב** = רפטביליות. אם יש באג בטבלאות — מתקנים ב-`.cls`, לא בכל מסמך מחדש.
